# Implementation Guide

## Objective

Deploy a production-ready static website using Amazon S3 and CloudFront.

## Prerequisites

- AWS Account
- Git
- HTML/CSS/JavaScript files
- IAM permissions

## Deployment Steps

### Step 1
Create a private S3 bucket.

### Step 2
Enable Versioning.

### Step 3
Upload website files.

### Step 4
Create CloudFront Distribution.

### Step 5
Configure Origin Access Control.

### Step 6
Attach Bucket Policy.

### Step 7
Test Deployment.
